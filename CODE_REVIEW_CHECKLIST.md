# Code Review Checklist

## Code Quality Standards

**Primary Check:**
```
make sure there are no dead code, all code blocks follow each other in order of implementation with a logical ierarchy, make sure all notes match the code
```

- **"checklist"** - Use this word to reference the full checklist

## Full Checklist Details

### ✅ Dead Code Check:
- [ ] No unused imports
- [ ] No unused variables/constants
- [ ] No unused functions/methods
- [ ] No commented-out code blocks

### ✅ Code Organization:
- [ ] Methods follow logical implementation order
- [ ] Related functions grouped together
- [ ] Proper MARK comments for sections
- [ ] Clear separation of concerns

### ✅ Comments & Documentation:
- [ ] All comments match actual code functionality
- [ ] No outdated or misleading comments
- [ ] Clear, concise documentation
- [ ] Proper code structure hierarchy
