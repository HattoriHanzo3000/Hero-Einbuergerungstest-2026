Hero: Einbürgerungstest — App Review Notes (v2.3)
=================================================
Paste into App Store Connect → App Review Information → Notes (or version-specific review notes).

---

TEST SCOPE

- Primary: onboarding → home → Smart Learning → Learn by Topics → Test Simulation → Cockpit → Settings.
- New in 2.3: learning progress syncs via the user’s iCloud account (CloudKit). No Hero login.
- Interface languages: German, English, Russian, Turkish, and Ukrainian (uk). Test Simulation questions remain in German (official exam language); UI chrome localizes with the selected app language.

ACCESS

- No account registration. Open the app and complete onboarding (federal state, optional test date, language). Choose Українська in onboarding or Settings → Personalisation to verify Ukrainian UI.
- Core learning works immediately on the free tier.
- Hero Pro: Settings → Hero Pro, or paywall entry points. Restore Purchases on the paywall.

iCLOUD SYNC (2.3)

- Progress (statistics, topic answers, favorites, federal state, test date) syncs automatically when the reviewer is signed in to iCloud on the test device.
- Container: iCloud.com.gizatech.Leben-in-Deutschland (private CloudKit database).
- Settings → Synchronization section explains sync; there is no in-app iCloud toggle — sync follows system iCloud.
- To verify: answer questions on one device, wait for sync, open the same Apple ID on a second device — progress should appear without reinstall.
- Federal state switch (Settings → Region): loads per-state progress; does not wipe other states.
- Settings → Reset App clears all learning data (all states).

SUBSCRIPTIONS

- Auto-renewable subscriptions + optional lifetime IAP via RevenueCat.
- Restore: paywall or Settings → Hero Pro → Restore Purchases.

QUESTION REPORTS (2.3)

- Flag icon on any question card opens an in-app report form (no Mail app).
- Requires iCloud sign-in; saves to CloudKit public database (container: iCloud.com.gizatech.Leben-in-Deutschland).
- Fields: question ID/text, report type, user message, optional email, app/device metadata.
- Reports are developer-only (CloudKit Console); not visible to other users.
- Without iCloud or network, submit shows a localized error; app does not crash.

NETWORK

- Question content is bundled (JSON). iCloud is used for personal progress sync (private database) and optional question reports (public database).
- App should not crash when iCloud is unavailable; progress remains local on device.

CONTACT

- Settings → Contact Us (in-app mail compose).
