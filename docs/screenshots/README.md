# README screenshots

Add four PNG files here (iPhone screenshots; portrait is fine).

Rename your files to these **exact** names:

| Your screen | File name |
|-------------|-----------|
| Home | `home.png` |
| Learn by topics | `learn-by-topics.png` |
| Progress (Cockpit) | `progress.png` |
| Search | `search.png` |

Then commit and push:

```bash
git add docs/screenshots/*.png
git commit -m "Add README screenshots"
git push origin develop
```

Images appear on the main [README](../../README.md) at the repo root.
